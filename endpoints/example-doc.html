Create User

Method: POST
Path: /api/users

Description

Creates a new user account in the system.

Input

{
  "username": "string",
  "email": "string",
  "password": "string",
  "isAdmin": false
}
username
Required. User’s unique handle (3–32 characters).
email
Required. Valid email address for account verification.
password
Required. Must be at least 8 characters; hashed before storage.
isAdmin
Optional. Boolean flag indicating administrative privileges. Defaults to false.

Output

{
  "id": "uuid",
  "username": "string",
  "email": "string",
  "isAdmin": false,
  "createdAt": "ISO-8601 timestamp"
}
id
Unique identifier for the user.
username
Echo of the provided username.
email
Echo of the provided email address.
isAdmin
Reflects the user’s admin status.
createdAt
UTC creation time of the record.
