Generate AI Canned Statements

Method: POST
Path: /api/ai/can

Description

Creates a given number of random canned AI statements.

Input

{
  "quantity": int
}
quantity
Required. Number representing number of statements to generate.

Output

{
  "response": int,
  "response-text": "string",
  "ai-canned-response-1": "string",
  "ai-canned-response-2": "string",
  "ai-canned-response-3": "string",
  "ai-canned-response-...": "string",
}
response
Response returned by the API request.
response-text
Response text returned by the API request.
ai-canned-response-1
The first AI canned response.
ai-canned-response-2
The second AI canned response.
ai-canned-response-3
The third AI canned response.
ai-canned-response-...
The number of canned responses will continue until the number is equal to the quantity given.
