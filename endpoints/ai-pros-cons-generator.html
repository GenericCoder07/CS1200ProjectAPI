Generate AI Pros and Cons List

Method: POST
Path: /api/ai

Description

Creates a pros and cons list for the given statement.

Input

{
  "msg": "string"
}
msg
Required. The statement that the AI will recieve.

Output

{
  "response": int,
  "response-text": "string",
  "ai-text": "string"
}
response
Response returned by the API request.
response-text
Response text returned by the API request.
ai-text
The pros and cons list generated by the AI.
